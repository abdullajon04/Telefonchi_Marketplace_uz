## 1. ПРЕДМЕТ ДОГОВОРА

1.1. Исполнитель обязуется разработать мобильное приложение «Mobile Market» — маркетплейс для торговли мобильными телефонами для платформы Android.

1.2. Заказчик обязуется предоставить необходимую информацию и содействие для реализации проекта.

1.3. Приложение включает следующие функции:
- Регистрация и авторизация пользователей (продавцы и покупатели)
- Каталог товаров с поиском и фильтрацией
- Корзина покупок и оформление заказов
- Панель управления для продавцов
- Статистика и аналитика продаж

## 2. ОБЯЗАННОСТИ СТОРОН

### 2.1. Исполнитель обязуется:
- Разработать приложение на основе утверждённого технического задания
- Предоставить исходный код и документацию
- Обеспечить функционирование приложения на устройствах Android 5.0+
- Провести тестирование приложения
- Передать права на использование приложения Заказчику

### 2.2. Заказчик обязуется:
- Предоставить необходимые данные о бизнес-процессах
- Обеспечить доступ к информации о товарах и категориях
- Содействовать в тестировании приложения
- Предоставить отзыв о выполненной работе

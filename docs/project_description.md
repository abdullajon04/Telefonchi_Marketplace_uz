# Описание проекта: Mobile Market

## Название проекта
**Mobile Market** — Мобильное приложение-маркетплейс для торговли мобильными телефонами

## Цель проекта
Разработка мобильного приложения для платформы Android, которое объединяет продавцов и покупателей мобильных телефонов на единой платформе. Приложение обеспечивает удобный процесс покупки и продажи товаров с современным и интуитивным интерфейсом.

## Целевая аудитория
- **Продавцы** — магазины мобильных телефонов, индивидуальные предприниматели
- **Покупатели** — физические лица, заинтересованные в покупке мобильных устройств и аксессуаров

## Аффилированная организация
Ассоциация предпринимателей района Джалакудук (Jalaquduq tumani tadbirkorlar uyushmasi)

## Основные функции

### Для продавцов
| Функция | Описание |
|---------|----------|
| Управление товарами | Добавление, редактирование и удаление товаров |
| Панель статистики | Просмотр общего количества товаров, заказов, продаж и выручки |
| Управление заказами | Просмотр заказов, изменение статуса (в обработке, отправлен, доставлен) |
| Профиль магазина | Управление данными магазина |

### Для покупателей
| Функция | Описание |
|---------|----------|
| Каталог товаров | Просмотр телефонов по категориям (смартфоны, кнопочные, планшеты, аксессуары) |
| Поиск и фильтрация | Поиск по названию, фильтрация по категории и цене |
| Корзина покупок | Добавление товаров, управление количеством |
| Оформление заказа | Выбор способа оплаты (онлайн/офлайн), указание адреса доставки |
| История заказов | Просмотр всех заказов и их статусов |
| Связь с продавцом | Возможность связаться через чат или телефон |

### Общие функции
| Функция | Описание |
|---------|----------|
| Регистрация | Регистрация с выбором роли (продавец/покупатель) |
| Авторизация | Вход по email/телефону и паролю |
| Безопасность | Валидация данных, проверка паролей |
| Профиль | Просмотр и управление личными данными |

## Категории товаров
1. **Смартфоны** — Samsung Galaxy, iPhone, Xiaomi и др.
2. **Кнопочные телефоны** — Nokia, Philips и др.
3. **Планшеты** — Samsung Galaxy Tab и др.
4. **Аксессуары** — чехлы, наушники, зарядные устройства

## Процесс работы приложения

### Регистрация и вход
1. Пользователь открывает приложение → видит экран приветствия
2. Выбирает роль: Продавец или Покупатель
3. Заполняет форму регистрации (имя, email, телефон, пароль, адрес)
4. Входит в систему через email/телефон и пароль

### Процесс покупки
1. Покупатель просматривает каталог или использует поиск
2. Выбирает товар и просматривает подробности
3. Добавляет товар в корзину
4. Оформляет заказ с указанием адреса и способа оплаты
5. Получает подтверждение заказа

### Процесс продажи
1. Продавец добавляет товар (название, описание, цена, категория, фото)
2. Товар появляется в общем каталоге
3. Получает заказы от покупателей
4. Управляет статусом заказов
5. Просматривает аналитику на панели управления
